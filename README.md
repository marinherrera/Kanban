## Sobre o projeto

Esse projeto é um quadro Kanban feito para organizar tarefas.
Os cards podem ser movidos entre as colunas.

---

## Funcionalidades

* Arrastar cards
* Organizar tarefas
* Layout responsivo

---

## Objetivo

Desenvolver um sistema Kanban simples para praticar a criação de interfaces interativas utilizando HTML, CSS e JavaScript.
